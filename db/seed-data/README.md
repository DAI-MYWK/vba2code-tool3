# マスタデータ準備ガイド

このディレクトリには、Excel ツールから移行するマスタデータを準備します。

## 必要なデータ

### 1. 掲載店舗マスタ (stores.json)

Excel の「掲載店舗」シートから 126 件の店舗情報を抽出してください。

形式:

```json
[
  {
    "name": "株式会社ホットスタッフ一宮",
    "branchCode": "[0901]"
  }
]
```

### 2. 抜きたい文言マスタ (removal_phrases.json)

Excel の「抜きたい文言」シートから 1,130 件の文言を抽出してください。

形式:

```json
[
  {
    "category": "福利厚生",
    "phrase": "WEBエントリー24時間受付中"
  }
]
```

カテゴリ例:

- 福利厚生
- PR・職場情報
- 応募資格
- その他

### 3. 非掲載フラグ文言 (ng_flag_phrases.json)

非掲載判定に使用するキーワードを抽出してください。

形式:

```json
[{ "phrase": "入社祝い金" }, { "phrase": "祝い金" }]
```

### 4. 年齢制限 NG 文言 (age_ng_phrases.json)

年齢表記の NG 文言を抽出してください。

形式:

```json
[{ "phrase": "歳以上の方大歓迎" }, { "phrase": "歳以下の方大歓迎" }]
```

## データ投入方法

1. 上記の JSON ファイルをこのディレクトリに配置
2. `db/seed.ts` を実行
3. または管理画面から CSV アップロードで投入

## 注意事項

- 最低賃金マスタは`db/seed.ts`に既に定義済みです
- データは年次更新が必要です
- バックアップを定期的に取得してください
